gitでのwebページの作り方（例）

超絶単純簡単なgitを使った、ウェブサイト作りのフローチャートみたいな

他のページを見てリポジトリを作ってあってあるのが前提。

  1. リモートのほうでworkブランチを作っておく。
  2. [git pull]でリモートに落とす
  3. [git branch -a]でさっき作ったリモートのworkがあるかチェック。
  4. [git checkout work]でローカルにworkを作り入る。
  5. [vi ファイル名.html]で新しくファイルを作成する。または、作ってあるそのファイルを開く。
  6. [git status]で改変したファイルが赤文字で表示されればおｋ。
  7. [git add ファイル名.html]でファイルをアドする。
  8. [git commit -m "コメント"]でコミットする。
  9. [git push]でリモートにあげる。
  10. リモートでプルリクエストしてマージする。そして、リモートのworkブランチを削除する。
  11. [git checkout master]でマスターブランチへ移る。
  12. [git merge --no-ff work]でローカルのworkをマスターにマージする。
  13. [git branch -d work]でローカルのworkブランチを削除する。
  14. [git branch]でworkが消えたかチェックする。

こんな感じでやっていく、なんかあったらpullしてみよう(´ω｀)ﾉｼ
