gitコマンド集

コマンド コマンドの動作
git init リポジトリの作成
git clone リモートのリポジトリを複製する。
git status 変更されたファイルを赤文字で表示する。
git diff ファイルの変更点を詳細に、行ごとに表示する。
git add [ファイル名] コミットするファイルを指定する。これがないとコミットできない。[git add .]で変更があったファイル全部を選択する。
git commit コッミッとする。[-m "コメント"]をつけるとエディタが開かない。
git log コミットのログが見える。
git reset 直前のコミットを取り消す。
git branch ブランチの一覧を表示。ブランチの作成。
git checkout [移動先のブランチ名] ブランチの移動
git merge [マージするブランチ名] ブランチをマージする。
git pull リモートリポジトリをローカルリポジトリに落とす。
git push リモートリポジトリにローカルリポジトリを同期する。

右も左もわからないなら、これくらいのコマンドさえ知っていれば基本は大丈夫だとおもうよ！
