gitコマンド集

コマンド コマンドの動作
git init リポジトリの作成
git clone リモートのリポジトリを複製する。
git status 変更されたファイルを赤文字で表示する。
git diff ファイルの変更点を詳細に、行ごとに表示する。
git add コミットするファイルを指定する。これがないとコミットできない。
git commit コッミッとする。[-m "コメント"]をつけるとエディタが開かない。
git log コミットのログが見える。
git reset 直前のコミットを取り消す。
git branch ブランチの一覧を表示。ブランチの作成。
git checkout ブランチの移動
