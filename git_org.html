gitを組織で使う

  • 組織の作り方

    • ウェブサイトの右上の自分のアイコンの左横の＋マークを押すと「New repository」を押す。
    組織の名前とメールアドレスを入力する。最後にその組織に招待したい人のアドレスかユーザーネームを入力して招待します。招待された人はメールから組織に加入します。

  • ピープル・チーム・コラボレイターの招待

    • 組織のリポジトリ画面のpeopleの下にある「Invite someone」をおして、追加したい人の名前かメールアドレスを入力して招待する。
    組織のsettings画面を開いて、左のCollaborators & teamsのページを開いてTeamsにある「create team」をおしてチームを作ると、自分がオーナーとして登録される。「add a person」に名前を入力して招待する。 チームを作った後に、そのチームを追加したいリポジトリのsettingsのteamの「add a team:」を押し、先ほど作ったチームを追加する。
    コラボレイターは共同編集者で、オーナーほど強い権限は持っていない。コラボレイターも名前かアドレスで検索して、招待する。

  • メンバーの権限
    • Admin : 管理者。クローン、プッシュのなどができ、コラボレイターを追加することができる。
    • Write : クローンされたリポジトリを開き、編集しプッシュまではできる。
    • Read : 読み取り以外できないようになっている。
